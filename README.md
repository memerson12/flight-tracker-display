# Flight Tracker Display

OLED-friendly flight tracker display with live telemetry and optional photo slideshow.

## Getting started

```sh
pnpm install
pnpm dev
```

The app runs on http://localhost:8080 and proxies API requests to the backend on port 8000.

## Backend service

```sh
cd backend
pnpm install
pnpm start
```

## Useful commands

```sh
pnpm build
pnpm lint
pnpm test
```
